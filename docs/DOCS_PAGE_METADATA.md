# Каталог документации: автоматизация

## Цели
- подключить все `.md` файлы из `docs/` к странице `src/pages/Docs.jsx` без ручного списка;
- исключить дублирование метаданных (title/description) между кодом и содержимым документа;
- обеспечить fallback по имени файла, если автор новой заметки не заполнил шапку.

## Источник данных
- Используем `import.meta.glob("../../docs/*.md", { eager: true, as: "raw" })` для подхвата всех файлов во время сборки Vite.
- Для каждого файла вычисляются поля `id`, `title`, `description`, `content`, `sourceName`, `searchText`.
  - `extractMetadata` анализирует первые строки и считывает `name:` и `description:` до первой пустой строки или текста.
  - `content` очищено от заголовка метаданных и хранит чистый Markdown.
  - `searchText` собирает `title`, `description`, исходное имя файла и содержимое для полнотекстового поиска.
- `DocCard` выводит `sourceName` (оригинальное имя файла) и скрывает описание, если оно отсутствует.
- В модальном окне отображается тот же `sourceName` и, при наличии, описание перед основным телом.

## Требования к файлам в docs/
- Первая строка: `name: <читаемое название карточки>`.
- Вторая строка: `description: <краткое описание карточки>`.
- Далее — пустая строка и основной Markdown-контент.
- Если метаданные не заданы, карточка показывает только имя файла (без `.md`) и не выводит description.
- Все существующие документы получили шапку, соблюдающую формат `name` + `description`.

## UI и UX
- Уточнены тексты заголовков, подсказок и состояний пустого поиска на странице Docs.
- Карточки получили бейдж с именем исходного файла и кнопку `Читать документ`.
- Добавлена статистика `Найдено документов` — число результатов после фильтрации.
- Ссылочные блоки с внешними ресурсами сохранены, стили приведены к единому виду с hover-состояниями.

## Проверено
- Поиск по названию, описанию и телу Markdown работает с учетом новых полей.
- Отсутствие метаданных (временное удаление шапки) показывает fallback с именем файла и без описания.
- Модальное окно корректно отображает текст документа без строки метаданных в начале.
