# Flipper Zero — Краткое ТЗ (UI + Admin + Content Guidelines + Firmware Structure)

## 1. UI — Пользовательский интерфейс
- Главная страница: обложка PNG, краткое описание, карточки блоков (базовые функции, кастомные прошивки, модули, гайды, уязвимости).
- Список категорий: колонки название/описание/число статей, фильтры по типу (basic/firmware/module/guide/vuln), карточки для mobile.
- Страница категории: заголовок, описание, список статей (карточки), сортировка по дате/названию.
- Страница статьи: заголовок, категория, тип, привязка к прошивке/модулю, Markdown→HTML контент, якоря секций.
- Страница прошивки: заголовок, описание, особенности, новые функции, связанные модули, статьи по прошивке.
- Модули: список с фильтрами; страница модуля — описание, для какой прошивки, пример использования.

## 2. Admin — Интерфейс управления
- Навигация блока Flipper Zero: Категории / Прошивки / Модули / Статьи / Очередь задач.
- Категории: CRUD, поля title/slug/type/parent_id/description/position, просмотр дерева.
- Прошивки: CRUD, поля name/slug/description_raw/rendered/repo_url/homepage_url, быстрый toggle активности.
- Модули: CRUD, поля name/slug/firmware_id/supported_firmwares/description/source_url.
- Статьи: таблица (название, категория, тип, статус, обновлено), фильтры (категория/тип/статус/прошивка), редактор summary/content_raw/tags, предпросмотр.
- Очередь задач: таблица задач, просмотр payload, статусы pending/processing/done/error, ручной перезапуск.

## 3. Content Guidelines — для AI/n8n
- Общие: чистый Markdown, без HTML, структурировано, без воды; вводная + 6–10 разделов + вывод.
- feature_basic: что это, как работает, где используется, как Flipper взаимодействует, ограничения, примеры, безопасные рекомендации.
- feature_custom_fw: описание, отличие от официальной, расширенные функции, поддерживаемые модули, ограничения, для кого подходит.
- module_custom: назначение, поддерживаемые прошивки, пример использования, ограничения, настройки.
- guide_scenario: интро, требования, шаги 1–N, чего нельзя, пример результата.
- vuln_check: тип уязвимости, описание, как протестировать, какая техника уязвима (абстрактно), меры защиты.

## 4. Firmware Structure — материалы по прошивкам
- Поддерживаемые: Official, Unleashed, Marauder (Wi-Fi Board), Momentum, BunnyLoader/BadUSB.
- Для каждой: полное описание, особенности, новые возможности, список модулей, связанные статьи.
- Минимум: отличие от официальной, технические изменения, список расширений, ограничения, способы установки.
- Дополнительно: таблица функций, таблица совместимости модулей.
