## 6. База уязвимостей

### Типы уязвимостей:

- Фиксированный код Sub-GHz
- Уязвимые NFC-метки
- Некорректная реализация BLE сервисов
- ИК устройства без проверок
- Некачественная реализация iButton

Каждая запись содержит:

- Тип
- Кратко
- Пример уязвимого устройства (обезличенно)
- Как протестировать

## 7. Очередь статей (интеграция с n8n)

Типы статей:

- `feature_basic`
- `feature_custom_fw`
- `module_custom`
- `guide_scenario`
- `vuln_check`

Поля очереди:

- `title`
- `type`
- `payload` (сырые данные)
- `status`: `draft` / `processing` / `ready` / `published` / `error`
