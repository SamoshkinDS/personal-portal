name: Ленивая загрузка роутов
description: Перевёл страницы SPA на React.lazy с единым fallback, чтобы ускорить первый рендер и уменьшить вес стартового бандла.

- 2025-11-20: заменены статические импорты на ленивые в `src/router.jsx`, добавлен общий загрузчик `src/components/PageLoader.jsx`.
- Теперь код каждой страницы подгружается по требованию, что убирает лишние dev-запросы к модулям и уменьшает размер initial chunk в продакшене.
- 2025-11-20: вынес тяжёлые библиотеки в отдельные чанки (`tiptap*`, `recharts`) через `manualChunks` в `vite.config.js` и обернул редакторы статей/заметок в ленивые компоненты (`PlantArticleEditorLazy`, `NotesEditorModal` через Suspense), чтобы tiptap и react-icons не попадали в стартовый vendor.
