name: Укрепление безопасности портала (ноябрь 2025)
description: Закрыли публичные маршруты, усилили проверку прав и добавили защиту от перебора паролей.

- API analytics и статьи теперь требуют права `view_analytics`; сторонний доступ к очереди по токену оставлен только для режима `processing`.
- Маршруты VPN/VLESS пускают только пользователей с `view_vpn` или флагом `vpn_can_create`; убран доступ обычных NON_ADMIN.
- JWT-секрет обязателен через переменную окружения (новый модуль `lib/jwt.js`), убран дефолтный ключ.
- `/api/auth/reset-password` защищён авторизацией и требует текущий пароль; добавлена rate-limit защита для логина/регистрации/проверки логина.
- В настройках появилась форма смены пароля; публичные формы сброса пароля переведены на защищённый маршрут.
- backend/.env дополнен примером `JWT_SECRET` для dev; добавлен middleware rateLimit.
- Дата: 22.11.2025.
