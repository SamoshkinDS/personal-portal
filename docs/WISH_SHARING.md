name: Публичный шэринг Wish List
description: Добавлен клиентский UI для выдачи временных публичных ссылок на список желаний с выбором срока действия и только правом просмотра.

- Страница `src/pages/WishList.jsx`: восстановлен основной функционал, добавлена кнопка «Поделиться» и модалка создания ссылки с пресетами сроков (1 час/день/неделя/месяц/до отключения), копирование сгенерированного URL `/shared/:token`.
- В карточках и таблице Wishlist сохраняется работа с архивом, редактированием, удалением и фильтрами; режим «Карточки/Таблица» снова работает.
- Публичные ссылки создаются через `sharedLinksApi.create` с `pageType: "wish-list"` и сохраняют только просмотр без навигации/CRUD.
