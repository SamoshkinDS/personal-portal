name: Workspace
description: Страница рабочего стола с настраиваемыми виджетами, авто-задачами и доской.

- Виджеты: список включает блок «До передачи показаний» (берёт дату из `/api/home/meter-settings`) и «Ближайшие оплаты» (данные из `/api/accounting/dashboard`), пользователь может включать/отключать их и порядок хранится в `workspace_settings`.
- Авто-задачи: вычисляются на основе ближайших оплат и дедлайна по счётчикам; выводятся отдельным столбцом слева от TaskBoard. В настройке задаётся горизонт напоминания (1–14 дней).
- Настройки: backend-роут `/api/workspace/settings` (таблица `workspace_settings`) хранит выбранные виджеты и `auto_task_lead_days`. Дата наполнения: 26.11.2025.
